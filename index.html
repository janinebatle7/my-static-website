Laboratory Activity No.1: Introduction to HTML5 Basics

It relies on tags or elements to organize and display webpage content.
Learning the basic HTML tags, proper document structure, and related concepts
such as DOM, XML, XHTML, and Web Storage APIs is important in web development.
